# 🏥 Hospital Management System

A full-stack Hospital Management System built with Laravel API and React.js.

---

## 🚀 Features

- Authentication (Login / Register)
- Dashboard with Statistics
- Doctors Management
- Patients Management
- Departments Management
- Appointments Management
- Prescriptions Management
- Medicines Management
- Activity Logs
- Dark Mode
- PDF Export
- Excel Export
- Responsive UI

---

## 🛠️ Technologies Used

### Backend
- Laravel 12
- MySQL
- Sanctum Authentication
- REST API

### Frontend
- React.js
- React Router
- Axios
- Bootstrap 5
- React Icons
- Chart.js
- React Toastify

---

## 📂 Project Structure

```
hospital-api/
hospital-frontend/
```

---

## ⚙️ Installation

### Backend

```bash
cd hospital-api

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```

### Frontend

```bash
cd hospital-frontend

npm install

npm run dev
```
